# Payam Kelich's Personal Website

## Description
This repository contains the source code for the personal website of Payam Kelich. It is a single-page, clean, and elegant personal portfolio that showcases Payam's profile, experience, publications, and contact information.

## Structure
- `index.html`, `experience.html`, `publications.html`: public site pages
- `assets/css/`: shared styles
- `assets/files/`: downloadable PDFs
- `assets/images/profile/`: profile and favicon images
- `assets/images/logos/`: institution and company logos
- `assets/images/publications/`: publication figures and cover images
- `scripts/`: local helper scripts
